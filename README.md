🦘 Hoppy QUIC

Hoppy — это современный, сверхбыстрый и устойчивый к цензуре SOCKS5-туннель, построенный на протоколе QUIC (UDP). Вдохновлен архитектурой Hysteria 2, но реализован на максимально производительном Rust.
├── src/
│   └── main.rs
├── Cargo.toml
├── install_vps.sh
└── README.md


✨ Основные преимущества

QUIC вместо TCP: Забудьте о задержках при потере пакетов (Head-of-Line blocking). Идеально для мобильного интернета.

Маскировка (SNI): Весь трафик выглядит как стандартный зашифрованный HTTP/3 поток к доверенным доменам.

Anti-DPI: Благодаря UDP-транспорту и TLS 1.3, провайдеру крайне сложно отличить туннель от обычного веб-трафика.

Zero-Config: Сервер автоматически генерирует сертификаты.

🚀 Быстрый старт (Сервер)

Для запуска на вашем VPS (Ubuntu/Debian):

Скачайте репозиторий: git clone https://github.com/ВАШ_НИК/hoppy.git && cd hoppy

Запустите скрипт установки: bash install_vps.sh

Убедитесь, что UDP порт 8844 открыт в фаерволе сервера.

💻 Использование (Клиент)

Скачайте файл для вашей ОС из раздела Releases вашего репозитория.

Запустите файл. Он автоматически откроется в интерактивном режиме.

Выберите режим 2 (Client) и введите IP вашего сервера.

Настройте ваш браузер или систему на SOCKS5 прокси: 127.0.0.1:1080.

🛠 Ручная сборка

cargo build --release


Бинарный файл появится в папке target/release/.
